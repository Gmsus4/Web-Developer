HTML: Formularios y Tablas

Table

Con la etiqueta <table> define una tabla.

Etiqueta <td>

Cada celda de la tabla está definida por una etiqueta <td> y </td> .

Etiqueta <tr>

Cada fila de la tabla comienza con una etiqueta <tr> y termina con una etiqueta </tr>.

Etiqueta <th>

A veces desea que sus celdas sean encabezados, en esos casos use la etiqueta <th> en lugar de la etiqueta <td>

Etiqueta <thead>

La etiqueta <thead> se usa para agrupar el contenido del encabezado en una tabla HTML.

<table>
	<thead><!-- Encabezado de la tabla -->
		  <tr>
		    <th>Person 1</th>
		    <th>Person 2</th>
		    <th>Person 3</th>
		  </tr>
	</thead>
  <tr>
    <td>Emil</td>
    <td>Tobias</td>
    <td>Linus</td>
  </tr>
  <tr>
    <td>16</td>
    <td>14</td>
    <td>10</td>
  </tr>
</table>

Etiqueta <tbody>

La etiqueta <tbody> se usa para agrupar el contenido del cuerpo en una tabla HTML.

<table>
	<thead><!-- Encabezado de la tabla -->
		  <tr>
		    <th>Person 1</th>
		    <th>Person 2</th>
		    <th>Person 3</th>
		  </tr>
	</thead>

	<tbody> <!-- Cuerpo de la tabla -->
		  <tr>
		    <td>Emil</td>
		    <td>Tobias</td>
		    <td>Linus</td>
		  </tr>
	</tbody>

  <tr>
    <td>16</td>
    <td>14</td>
    <td>10</td>
  </tr>
</table>

Etiqueta <tfoot>

La etiqueta <tfoot> se utiliza para agrupar el contenido del pie de página en una tabla HTML.

<table>
	<thead><!-- Encabezado de la tabla -->
		  <tr>
		    <th>Person 1</th>
		    <th>Person 2</th>
		    <th>Person 3</th>
		  </tr>
	</thead>

	<tbody> <!-- Cuerpo de la tabla -->
		  <tr>
		    <td>Emil</td>
		    <td>Tobias</td>
		    <td>Linus</td>
		  </tr>
	</tbody>
	
	<tfoot> <!-- Pie de página de la tabla -->
		  <tr>
		    <td>16</td>
		    <td>14</td>
		    <td>10</td>
		  </tr>
	</tfoot>
</table>

Form

La etiqueta <form> es un contenedor que contendrá un montón de controles de formulario individuales como entrada de texto y entradas de contraseñas y casillas de verificación.

El elemento Form también dicta dónde se deben enviar los datos del formulario cuando se envía dicho formulario.

El atributo action especifica a donde se debe enviar los datos.

Input

La etiqueta <input> especifica un campo de entrada donde el usuario puede ingresar datos.

El elemento <input> es el elemento de formulario más importante.

El elemento <input> se puede mostrar de varias formas, según el atributo de tipo.

<input type="">

Los diferentes tipos de entrada son los siguientes:

  • <input type="button"> Define un botón en el que se puede hacer clic (se usa principalmente con JavaScript para activar un script).
  • <input type="checkbox"> Define una casilla de verificación.
  • <input type="color"> Define un selector de color.
  • <input type="date"> Define un control de fecha (año, mes, día (sin hora)).
  • <input type="datetime-local"> Define un control de fecha y hora (año, mes, día, hora (sin zona horaria).
  • <input type="email"> Define un campo para una dirección de correo electrónico.
  • <input type="file"> Define un campo de selección de archivos y un botón "Examinar" (para cargas de archivos)
  • <input type="image"> Define una imagen como botón de envío.
  • <input type="month"> Define un control de mes y año (sin zona horaria).
  • <input type="number"> Define un campo para ingresar un número.
  • <input type="password"> Define un campo de contraseña.
  • <input type="radio"> Define un botón de radio.
  • <input type="range"> Define un control de rango (como un control deslizante).
  • <input type="reset"> Define un botón de reinicio.
  • <input type="search"> Define un campo de texto para ingresar una cadena de búsqueda.
  • <input type="submit"> Define un botón de envío.
  • <input type="tel"> Define un campo para ingresar un número de teléfono.
  • <input type="text"> (default value).
  • <input type="time"> Define un control para ingresar una hora (sin zona horaria).
  • <input type="url"> Define un campo para ingresar una URL.
  • <input type="week"> Define un control de semana y año (sin zona horaria).

Label

El elemento <label> representa una etiqueta que puede ser asociada a un control de formulario, y que se supone provee una descripción corta para éste. Los navegadores pueden enlazar ambos elementos permitiendo que los usuarios establezcan el enfoque sobre el control al hacer clic en su etiqueta.

<label for="name">Escribe tu nombre: </label>
<input type="text" id="name">

Buttons

Si tienen un botón dentro de un formulario, el comportamietno predeterminado será enviar ese formulario.

Pero tenemos una opción que podemos pasar en un atributo llamado type.

<button type="button">Enviar</button> <!--Se comportará como un botón-->
<!--Esto le dice al navegador que quiero que este botón se comporte como un botón,
no un botón de envío, sino solo un botón antiguo normal que no hará nada.
-->

<button type="submit"> <!-- Valor por defecto -->

Name Attribute

Especifica el nombre de un elemento.

<form action="">
  <label for="name">Name:</label>
  <input type="text" id="name" name="user-name"><!----->
</form>

Al agregar el atributo name, a la hora de enviar un formulario, tendrá en cuenta el name y además lo mostrará con su respectivo valor.

Text Area

El elemento HTML <textarea> representa un control para la edición mutilínea de texto sin formato.

<textarea> </textarea>

Validaciones en los Formularios

Se refiere a agregar restricciones o validar la entrada o los datos del usuario, por ejemplo, requerir que algunos campos no puedan estar vacíos o que una contraseña tenga que estar entre, 12 o 20 caracteres por ejemplo.

Uno de los atributos que permiten de alguna manera "Validar los formularios" es required.

<input type ="email" required>

Al agregar ese atributo, hace que sea requerido para el formulario, por lo contrario no podrá ser enviado.
