Bootstrap

¿Qué es Bootstrap?

Bootstrap nos ayuda a crear rápidemante sitios web modernos atractivos y receptivos para que no tengamos que escribir con tanto CSS como podríamos.

Consiste en una colección de componentes, diferentes piezas que podemos usar y simplemente usar en nuestra aplicación.

Container

Los contenedores son un bloque de construcción fundamental de Bootstrap que contienen, rellenan y alinean su contenido dentro de un dispositivo o ventana gráfica determinada.

Bootstrap viene con tres contenedores diferentes:

  • .container , Que establece un max-width en cada punto de interrupción de respuesta
  • .container-fluid, que está width: 100% en todos los puntos de interrupción
  • .container-{breakpoint}, que es width: 100% hasta el punto de interrupción especificado

Buttons

Las .btn clases están diseñadas para usarse con el <button> elemento. Sin embargo, también puede usar estas clases en elementos <a> o <input> (aunque algunos navegadores pueden aplicar una representación ligeramente diferente).

Grid

Las columnas tiene por defecto un espacio de 12.

Al poner la clase col crea un espacio de columna adaptable.

Si tenemos dos columnas en el row, con la clase col, las dos columnas tendrán un ancho del 50% esto porque se adaptan, al igual que si hubiera 10 columnas con la clase col, estas tendrían un ancho del 10%, ambas se adaptarían.

Si al poner la clase col-4 le agregamos un "-n" un guión seguido de un número, esto indicará que esa columna tomará (n) espacios del total que son 12.

En este caso tomaría 4 espacios de 12.

Los row crean filas.

<div class="container">
  <h1>Sistema de Grid</h1>
 
	<div class="row"> <!--Crea las Filas-->
	    <div class="col-2"></div> <!--Crea las columnas-->
	    <div class="col"></div> <!--Crea las columnas-->
  </div>

  <div class="row"> <!--Crea las Filas-->
			<div class="col-2"></div> <!--Crea las columnas-->
			<div class="col"></div> <!--Crea las columnas-->
  </div>
</div>

Navs

Documentación y ejemplos para el potente encabezado de navegación receptivo de Bootstrap, la barra de navegación. Incluye soporte para la marca, navegación y más, incluido el soporte para nuestro complemento de colapso.

Icons

Biblioteca de íconos de código abierto, de alta calidad y gratuita con más de 1300 íconos. Inclúyelos como quieras: SVG, SVG sprite o fuentes web. Úselos con o sin Bootstrap en cualquier proyecto.

Border

Utilice las utilidades de borde para diseñar rápidamente el borde y el radio del borde de un elemento. Ideal para imágenes, botones o cualquier otro elemento.

Shadows

Agregue o elimine sombras a los elementos con las utilidades de sombra de cuadro.

Spacing

Bootstrap incluye una amplia gama de clases de utilidad abreviada de margen de respuesta, relleno y espacio para modificar la apariencia de un elemento.

Carousel

El carrusel es una presentación de diapositivas para recorrer una serie de contenido, construido con transformaciones CSS 3D y un poco de JavaScript. Funciona con una serie de imágenes, texto o marcado personalizado. También incluye soporte para controles e indicadores anteriores / siguientes.

Dropdowns

Cambie las superposiciones contextuales para mostrar listas de enlaces y más con el complemento desplegable Bootstrap.

Spinners

Indique el estado de carga de un componente o página con spinners Bootstrap, construido completamente con HTML, CSS y sin JavaScript.
