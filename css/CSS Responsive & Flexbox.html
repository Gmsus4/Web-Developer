CSS: Responsive & Flexbox

¿Qué es Flexbox?

Es una serie de propiedades que usamos para diseñar elementos en nuestra página en una caja de contenido.

Tenemos algún contenedor y queremos distribuir el espacio dentro de ese contenedor.

Display Flex

<div class="contenedor">
		<div></div> items
		<div></div> items
		<div></div> items
		<div></div> items
		<div></div> items
</div>
.contenedor{ /*En el objeto padre es donde se aplica el Display Flex*/
	display: flex;
}

.contenedor div{
	width: 200px; /*Declaramos una ancho y alto*/
	height: 200px;
}

En el ejemplo, los divs empezarán de izquierda a derecha.

El eje principal

El eje principal está definido por flex-direction, que posee cuatro posibles valores:

  • row (Por default)
  • row-reverse
  • column
  • column-reverse

Flex Direction

flex-direction nos permite decidir la dirección del eje principal en nuestro contenedor.

.contenedor{ /*En el objeto padre es donde se aplica el Display Flex*/
	display: flex;
	flex-direction: row; /*De izquierda a derecha (Por defecto) - Horizontal*/
	flex-direction: row-reverse; /*Lo mismo que row, pero empieza desde el último*/
	flex-direction: column; /*Empieza el contenido de arriba hacia abajo - Vertical*/
	flex-direction: column-reverse; /*Lo mismo, pero empieza desde el último*/
}

Justify Content

justify-content determina cómo se distribuye el contenido en el eje principal.

.contenedor{ /*En el objeto padre es donde se aplica el Display Flex*/
	display: flex;
	flex-direction: row; /*De izquierda a derecha (Por defecto) - Horizontal*/
	justify-content: flex-start; /*Por defecto, de izquierda a derecha*/
	justify-content: flex-end; /*El contenido se desplazará al lado de la derecha*/
	justify-content: center; /*Lo centra a lo largo del eje principal*/
	justify-content: space-between; /*Distribuye todo el espacio del contenido, 
																		excepto los bordes exteriores*/
	justify-content: space-around; /*Distribuye el espacio en partes iguales en 
																		ambos lados*/
	justify-content: space-evenly; /*Distribuye el espacio de manera uniforme*/
}
.contenedor{ /*En el objeto padre es donde se aplica el Display Flex*/
	display: flex;
	flex-direction: column; /*De arriba hacia abajo (Por defecto) - Vertical*/
	justify-content: flex-start; /*Por defecto, de arriba hacia abajo*/
	justify-content: flex-end; /*El contenido se desplazará hacia abajo*/
	justify-content: center; /*Lo centra a lo largo del eje principal*/
	justify-content: space-between; /*Distribuye todo el espacio del contenido, 
																		excepto los bordes exteriores*/
	justify-content: space-around; /*Distribuye el espacio en partes iguales en 
																		ambos lados*/
	justify-content: space-evenly; /*Distribuye el espacio de manera uniforme*/
}

Flex Wrap

La propiedad flex-wrap de CSS especifica si los elementos "hijos" son obligados a permanecer en una misma línea o pueden fluir en varias líneas. Si la cobertura (wrap) está permitida, esta propiedad también te permite controlar la dirección en la cual serán apilados los elementos.

.contenedor{ /*En el objeto padre es donde se aplica el Display Flex*/
	display: flex;
	flex-direction: row;
	justify-content: flex-start;
	flex-wrap: nowrap; /* Default, todos los elementos flexibles estarán en una línea*/
	flex-wrap: wrap; /*Los elementos flexibles se envolverán en varias líneas, 
										de arriba a abajo.*/
	flex-wrap: wrap-reverse; /*los elementos flexibles se envolverán en varias 
															líneas de abajo hacia arriba.*/
}

Align items

Align-items, distribuirá los articulos a lo largo del eje transversal.

Esto define el comportamiento predeterminado de cómo se distribuyen los elementos flexibles a lo largo del eje transversal en la línea actual. Piense en ello como la versión de contenido justificado para el eje transversal (perpendicular al eje principal).

.contenedor{
	display: flex;
	flex-direction: row;
	justify-content: center;
	align-items: flex-start; /*Empieza desde arriba*/
	align-items: flex-end; /*Empieza desde abajo*/
	align-items: flex-center; /*Centra los items*/
	align-items: flex-strech; /*Estira los items hasta tomar la aultura máxima*/
	align-items: flex-baseline; /*Toma como referencia el punto central de un item
															 y los mantiene a raya*/
}

Align-content

Ajusta las líneas dentro de un contenedor flex cuando hay espacio extra en el eje transversal.

.contenedor{
	display: flex;
	flex-direction: row;
	justify-content: center;
	flex-wrap: wrap;

	/* Ajusta las líneas desde el inicio del eje transversal */
	align-content: flex-start;
	
	/* Ajusta las líneas desde el final del eje transversal */
	align-content: flex-end;
	
	/* Ajusta las líneas al rededor del centro del eje transversal */
	align-content: center;
	
	/* Distribuye las líneas a lo largo del eje transversal, de principio a fin */
	align-content: space-between;
	
	/* Distribuye las líneas a lo largo del eje transversal, igualmente espaciados */
	align-content: space-around;
	
	/* Estira las líneas para que ocupen el eje transversal completo */
	align-content: stretch;

Align Self

Esto permite (align-self) que se anule la alineación predeterminada (o la especificada por align-items) para elementos flexibles individuales.

.item {
  align-self: auto | flex-start | flex-end | center | baseline | stretch;
}

Flex Basis

flex-basis Determinará el tamaño inicial de un elemento antes de que se coloque en una especie de punto de partida antes de agregarlo a nuestro contenedor flexible.

Es solo el tamaño inicial en el que se debe agregar un elemento a nuestra caja y podría ser la altura dependiendo de la dirección del eje principal.

Flex Grow

flex-grow va a controlar la cantidad de espacio que ocupa un elemento si hay espacio disponible.

Es solo el tamaño inicial en el que se debe agregar un elemento a nuestra caja y podría ser la altura dependiendo de la dirección del eje principal.

div{
	flex-grow: 1; /*A todos los items se les otorgará un crecimiento igual, 
								tomando todo el espacio disponible*/
}

/*---------------------------*/


div:nth-of-type(1){
		flex-grow: 1;
}
div:nth-of-type(2){
		flex-grow: 1;
}
div:nth-of-type(3){
		flex-grow: 1;
}

/*---------------------------*/

div:nth-of-type(1){
		flex-grow: 1;
}
div:nth-of-type(2){
		flex-grow: 2;
}
div:nth-of-type(3){
		flex-grow: 1;
}

Flex Shrink

Especifica el factor de contracción de un flex item. Los flex items se encogerán para llenar el contenedor de acuerdo a su número flex-shrink , cuando el tamaño por defecto de los flex items sea mayor al de su contenedor flex container.

Flex

Esta es la abreviatura de flex-grow, flex-shrink y flex-base combinados. El segundo y tercer parámetro (flex-shrink y flex-base) son opcionales.

.item {
  flex: none | [ <'flex-grow'> <'flex-shrink'>? || <'flex-basis'> ]
}

¿Qué es Responsive Desing?

Responsive Desing se refiere a crear sitios web que puedan responder al dispositivo en el que está, ya sea el tamaño de la pantalla, si está en un dispositivo pequeño como un teléfono móvil o un monitor super grande o la orientación del dispositivo.

El objetivo es crear un sitio web único que responda a algo en el dispositivo en el que se muestra.

Media Queries

Nos permite cambiar nuestros estilos o agregar nuevos estilos dependiendo de algún parámetro dependiendo del tamaño de la pantalla de dicho dispositivo.

@media (min-width: 900px){ /*Si la pantalla tiene 900px o más, hará los cambios*/
	body{
			background-color: red;
	}
}


@media (max-width: 900px){ /*Si la pantalla tiene 900px o menos, hará los cambios*/
	body{
			background-color: blue;
	}
}


/*Si la pantalla está entre 600px y 900px*/
@media (min-width: 600px) and (max-width: 900px){ 
		body{
				background-color: black;
		}
}


/*Si giras la pantalla*/
@media (orientation: landscape){
		body{
				background-color: green;
		}
}
